Profile photo
=============

Put your photo in this folder and name it exactly:

    profile.jpg

Recommendations:
- Square image (same width and height), e.g. 600 x 600 pixels
- A clear, friendly headshot
- JPG format

Until you add the photo, the website shows a blue placeholder with
your initials (AML), so the page always looks complete.

If your photo is a PNG instead of a JPG:
- name it  profile.png
- open index.html and change  images/profile.jpg  to  images/profile.png
